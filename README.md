# vim开发环境

## 安装的插件
+ nerdtree(https://github.com/preservim/nerdtree.git)
+ tagbar(https://github.com/preservim/tagbar.git)
+ nerdcommenter(https://github.com/preservim/nerdcommenter.git)
+ vim-go(https://github.com/fatih/vim-go.git)

## 使用
wget https://raw.githubusercontent.com/ntt360/vim/master/vim8_setup.sh -O ~/vim8_setup.sh && sh ~/vim8_setup.sh && rm -rf ~/vim8_setup.sh
